# CNL 超能力插件套件

一个为 Minecraft 1.21.11 设计的超能力插件套件，包含 52 种独特的超能力、大逃杀模式游戏、客户端 Mod 按键触发。

> **GitHub 仓库：** [https://github.com/yeyue1413/cnl](https://github.com/yeyue1413/cnl)

---

## 组件说明

本仓库包含 **四个组件**：

| 组件 | 类型 | 说明 |
|------|------|------|
| `Abilities-1.0.0.jar` | 服务端插件 | 核心超能力插件，提供 52 种超能力 |
| `Battle-1.0.0.jar` | 服务端插件 | 大逃杀模式扩展，在超能力基础上提供大逃杀玩法 |
| `SuperpowerKeysBridge-1.0.0.jar` | 服务端插件 | 按键桥接插件，接收客户端 Mod 的按键消息并触发技能 |
| `superpower-keys-1.0.0.jar` | 客户端 Mod (Fabric) | 为客户端添加 3 个自定义按键绑定，按键消息通过数据包发送至服务端触发技能 |

---

## 环境要求

- **服务端**: Paper 1.21.11 或兼容版本
- **客户端**: Minecraft 1.21.11 + Fabric Loader >=0.16.0 + Fabric API
- **Java**: >= 21

---

## 安装方法

### 服务端安装

1. 将 `Abilities-1.0.0.jar` 放入服务端 `plugins/` 目录
2. 将 `Battle-1.0.0.jar` 放入服务端 `plugins/` 目录
3. 将 `SuperpowerKeysBridge-1.0.0.jar` 放入服务端 `plugins/` 目录
4. 重启服务端（或使用 plugman 加载）

### 客户端安装（必装）

> **注意：本插件的绝大多数主动技能必须安装客户端 Mod 才能使用。** 所有主动技能触发都依赖 Mod 向服务端发送按键数据包，不装 Mod 则技能无法触发。

1. 安装 Fabric Loader（>=0.16.0）
2. 将 `superpower-keys-1.0.0.jar` 放入客户端 `.minecraft/mods/` 目录
3. 确保同时安装了 Fabric API
4. 启动游戏

## 快速上手

### 1. 获得超能力

管理员通过以下命令为玩家分配超能力：

```
/cnl set <ID|random> [玩家]
/cnl random [玩家|@a]
```

示例：

```
/cnl set 1           给自己分配闪电侠（ID=1）
/cnl set random      给自己随机分配
/cnl random @a       所有在线玩家随机分配
```

### 2. 查看超能力

```
/cnl                 查看当前超能力
/cnl list            列出所有超能力
/cnl info <ID>       查看指定超能力的详情
```

### 3. 使用主动技能

- 安装客户端 Mod 后，在 `选项 → 控制 → 超能力按键` 中为 3 个技能键设置自定义按键
- 按下已绑定的按键即可触发技能
- 部分能力支持多个技能槽位，触发不同的功能

## 按键绑定说明（客户端 Mod）

安装客户端 Mod 后，游戏设置中会出现 3 个自定义按键，**默认均为未绑定**，需要手动设置：

| 按键 | 默认键位 | 设置位置 |
|------|----------|----------|
| 技能键 1 | 未绑定 | 选项 → 控制 → 超能力按键 |
| 技能键 2 | 未绑定 | 选项 → 控制 → 超能力按键 |
| 技能键 3 | 未绑定 | 选项 → 控制 → 超能力按键 |

玩家需要自行在控制设置中为每个技能键绑定一个按键（如 `P`、`G`、`V` 等），绑定完成后才能在游戏中使用。

## 大逃杀模式

配合 Battle 插件，可以在超能力基础上进行大逃杀玩法。

### 游戏流程

1. 管理员在等待大厅配置好队伍/模式等，点击全息按钮 **「开始游戏」** 启动对局
2. 所有生存/冒险模式的玩家自动成为参与者，传送至主世界
3. 聊天栏出现 **3 个随机超能力选项**，输入 **1 / 2 / 3** 选择能力（选完后自动确认）
4. 也可输入 **ok** 直接确认（未选能力则无能力参战）
5. 全部确认后 **5 秒倒计时**，倒计时结束开战
6. 安全圈逐圈缩小，圈外扣血
7. 最后存活的玩家/队伍获胜

### 等待大厅

游戏开始前，玩家会被传送到 `battle_lobby` 世界的大厅中等待。

**大厅功能：**
- 大厅为一个带有围墙的基岩平台，中间为准备区域
- OP 可点击全息按钮进行管理操作，包含 8 个按钮：
  - **同队同出生点** — 切换同队出生点开关
  - **特殊模式** — 切换永昼/永夜/正常模式
  - **随机分队** — 随机分配队伍
  - **切换组队** — 切换队伍人数模式
  - **开始游戏** — 启动对局
  - **超能力大师** — 切换大师模式
  - **重置世界** — 一键重置地图并重启服务器
  - **友伤** — 切换友好伤害开关
- 玩家可 **潜行 + F** 键打开染料界面手动选择队伍
- 大厅同时显示公告信息和作者信息

### 关于重置世界

`/cnl battle reset` 一键删除世界文件并重启服务器。

**执行流程：** 停游戏 → 踢玩家 → 删世界文件 → 生成 `restart.flag` → 关服

**自动重启原理：** 服务端需配合循环启动脚本。插件关服前生成 `restart.flag`，脚本检测到该文件则重新启动，否则退出。

```bash
while true; do java -jar $(ls paper-*.jar | head -1) nogui; [ -f restart.flag ] && rm restart.flag || break; done
```

> 不使用循环启动脚本时，执行重置后服务器将直接关闭，不会自动重启。

## 特殊机制说明

### 死亡重置（DeathReset）

- **开启**：死亡后随机获得新超能力
- **关闭**：死亡后保留当前超能力
- 玩家可通过 `/cnl deathreset` 查看状态
- 管理员可通过 `/cnl deathreset toggle/on/off` 控制

### 技能键系统

- **技能键1**：主要技能触发（需在控制设置中手动绑定按键）
- **技能键2**：辅助技能触发（需在控制设置中手动绑定按键）
- **技能键3**：附加功能触发（需在控制设置中手动绑定按键，如灵魂召唤师的释放全部灵魂）

### 隐藏能力

| ID | 名称 | 效果 |
|----|------|------|
| 100 | 飞雷神 | 受到伤害时免疫并瞬移，冷却 0.5 秒（不参与随机分配） |

## 常见问题

**Q: 必须安装客户端 Mod 吗？**  
A: **必须安装。** 绝大多数主动技能的触发都依赖客户端 Mod 通过数据包向服务端发送按键消息。少数能力（如锻造师、富豪、食矿者等通过副手右键、神之左手通过空手左键等交互方式触发）可以不装 Mod 使用，但大多数主动技能（如射击者、时停者、火焰人、雷电法王等）均需要 Mod 才能生效。

**Q: 技能键默认是什么？**  
A: 没有默认按键。3 个技能键全部是未绑定状态，需要在 `选项 → 控制 → 超能力按键` 中手动设置。

**Q: 超能力可以自己选吗？**  
A: 可以，管理员使用 `/cnl set <ID> [玩家]` 指定即可。

**Q: 大逃杀模式和普通模式冲突吗？**  
A: 不冲突。大逃杀模式在超能力基础上增加了圈缩、队伍、胜负判定等机制，普通模式仅使用超能力。

**Q: 这个项目发布源码吗？**  
A: **发布。** 源码已包含在本仓库的 `source code/` 目录中，支持自行构建和二次开发。
